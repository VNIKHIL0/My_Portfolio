🔐 Basic User Authentication System

🧩 Problem: Many modern apps need secure user management systems for login, registration, and route protection. Building a lightweight yet secure solution is essential for small and mid-scale projects.

🎯 Objective: To create a simple, frontend-backend integrated authentication system with JWT, password hashing, protected routes, and a responsive UI.

⚙️ Features

  • 🔐 Sign Up / Login / Logout
  • 🔑 JWT-based authentication
  • 🧂 Password hashing using bcrypt
  • 🛡️ Protected dashboard route
  • 🌐 CORS enabled for frontend-backend communication

🛠 Technology Stack

  • Frontend: HTML, Tailwind CSS, Vanilla JS
  • Backend: Node.js, Express.js
  • Database: MongoDB
  • Security: bcrypt, JWT
  • Deployment: Render (Backend), Netlify (Frontend)

🧪 API Routes

  • POST /auth/signup: Register new user
  • POST /auth/login: Authenticate and get token

🔐 JWT Workflow

  • User logs in → receives JWT
  • JWT stored in `localStorage`
  • Protected pages check token → if invalid, redirect to login
  • Logout clears the token
